What it does: Returns a friendly greeting message

What it does: Processes form data and returns a confirmation

POST /api/submit

💡 Expects: email, message

3️⃣ GET /api/time

What it does: Returns current server time and timestamp

GET /api/time

💡 No parameters needed

4️⃣ POST /api/calculate

What it does: Performs simple math operations

POST /api/calculate

💡 Expects: num1, num2, operation (add/subtract/multiply/divide)

5️⃣ GET /api/stats

What it does: Returns usage statistics and server metrics
